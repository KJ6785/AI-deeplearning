# 핵심 수식 & 계산 예시 (표준 버전)

## 1️⃣ **kpc (킬로파섹)**

### 정의
```
1 kpc = 1 kiloparsec = 3,086 광년 = 3.086 × 10¹⁹ m
```

### v와의 연관
```
v = 회전 속도 (km/s)
r = 은하 중심으로부터 거리 (kpc)

물리적 의미: 
- r이 클수록 은하 외곽
- v(r) = 은하 회전 곡선 (거리에 따른 속도 변화)
```

**전형적인 은하 회전 곡선:**
```
r = 2 kpc   → v = 120 km/s  (상승 구간)
r = 8 kpc   → v = 220 km/s  (피크)
r = 15 kpc  → v = 215 km/s  (평평한 구간)
r = 30 kpc  → v = 210 km/s  (평평 유지)
```

## 2️⃣ **에너지 보존 수식**

### 기본 공식
```
E = KE + PE = (1/2)*m*v² - G*M*m/r

KE = 운동 에너지 = (1/2)*m*v²
PE = 위치 에너지 = -G*M*m/r

m = 별의 질량 (kg)
v = 회전 속도 (m/s)
G = 6.674 × 10⁻¹¹ m³/(kg·s²)
M = 은하 질량 (kg)
r = 반지름 (m)
```

### 단위 질량당 에너지 (m으로 나눔)
```
E/m = (1/2)*v² - G*M/r

단위: (m/s)² = m²/s²
```

### 에너지 보존 위반 계산
```
방법 1 (표준편차 기반):
Violation = σ(E) / |E_avg|
σ(E) = √[Σ(Eᵢ - E_avg)² / n]

방법 2 (범위 기반):
Violation = (E_max - E_min) / |E_avg|

방법 3 (RMS):
Violation = √[Σ(Eᵢ - E_avg)² / n] / |E_avg|
```

### 계산 예시 (실제 은하 NGC 3198 스타일)
```
G = 6.674 × 10⁻¹¹ m³/(kg·s²)
M = 1.5 × 10¹¹ M_sun = 3 × 10⁴¹ kg

위치 1: r₁ = 5 kpc = 1.543 × 10²⁰ m, v₁ = 150 km/s = 1.5 × 10⁵ m/s
  KE₁ = (1/2)*(1.5×10⁵)² = 1.125 × 10¹⁰ m²/s²
  PE₁ = -(6.674×10⁻¹¹ × 3×10⁴¹) / (1.543×10²⁰) = -1.298 × 10¹¹ m²/s²
  E₁ = 1.125×10¹⁰ - 1.298×10¹¹ = -1.186 × 10¹¹ m²/s²

위치 2: r₂ = 10 kpc = 3.086 × 10²⁰ m, v₂ = 200 km/s = 2.0 × 10⁵ m/s
  KE₂ = (1/2)*(2.0×10⁵)² = 2.0 × 10¹⁰ m²/s²
  PE₂ = -(6.674×10⁻¹¹ × 3×10⁴¹) / (3.086×10²⁰) = -6.489 × 10¹⁰ m²/s²
  E₂ = 2.0×10¹⁰ - 6.489×10¹⁰ = -4.489 × 10¹⁰ m²/s²

위치 3: r₃ = 20 kpc = 6.172 × 10²⁰ m, v₃ = 210 km/s = 2.1 × 10⁵ m/s
  KE₃ = (1/2)*(2.1×10⁵)² = 2.205 × 10¹⁰ m²/s²
  PE₃ = -(6.674×10⁻¹¹ × 3×10⁴¹) / (6.172×10²⁰) = -3.244 × 10¹⁰ m²/s²
  E₃ = 2.205×10¹⁰ - 3.244×10¹⁰ = -1.039 × 10¹⁰ m²/s²

에너지 보존 체크:
E_avg = (-1.186×10¹¹ - 4.489×10¹⁰ - 1.039×10¹⁰) / 3 = -5.796 × 10¹⁰ m²/s²
E_max = -1.039 × 10¹⁰
E_min = -1.186 × 10¹¹
Range = 1.186×10¹¹ - 1.039×10¹⁰ = 1.082 × 10¹¹

Violation = 1.082×10¹¹ / 5.796×10¹⁰ = 1.87
```
**해석:** 에너지가 평균 대비 187% 변동 → 보존 위반 심각

## 3️⃣ **MOND 수식**

### 표준 MOND (Milgrom 1983)
```
기본 원리:
a_obs = μ(a_N/a₀) * a_N

a_obs = 관측 가속도
a_N = 뉴턴 가속도 = G*M/r²
a₀ = 1.2 × 10⁻¹⁰ m/s² (MOND 임계 가속도)
μ(x) = 보간 함수
```

### 보간 함수
```
단순 형태:
μ(x) = x / (1 + x)  (x = a_N/a₀)

고가속도 (a_N >> a₀): μ ≈ 1 → a_obs = a_N (뉴턴 역학)
저가속도 (a_N << a₀): μ ≈ x → a_obs = √(a_N * a₀)
```

### 회전 속도 공식
```
원운동: a = v²/r

저가속도 영역 (은하 외곽):
v²/r = √(G*M/r² * a₀)
v⁴ = G*M*a₀*r
v = (G*M*a₀)^(1/4) * r^(1/4)

또는 간단히:
v = (G*M*a₀)^(1/4) (거리 무관, 평평한 회전 곡선!)
```

### 계산 예시
```
은하 질량: M = 5 × 10¹⁰ M_sun = 1 × 10⁴¹ kg
a₀ = 1.2 × 10⁻¹⁰ m/s²
G = 6.674 × 10⁻¹¹ m³/(kg·s²)

v = (G*M*a₀)^(1/4)
v = (6.674×10⁻¹¹ × 1×10⁴¹ × 1.2×10⁻¹⁰)^(1/4)
v = (8.009×10²⁰)^(1/4)
v = 1.68 × 10⁵ m/s = 168 km/s

모든 거리에서 동일:
r = 5 kpc  → v = 168 km/s
r = 10 kpc → v = 168 km/s
r = 20 kpc → v = 168 km/s
```

### 바리온 물질 포함 (실용 버전)
```
v²_total = v²_baryon + v²_MOND

v_baryon = √(G*M_baryon/r)  (내부 영역)
v_MOND = (G*M_baryon*a₀)^(1/4)  (외부 영역)

예시:
M_baryon = 5×10¹⁰ M_sun
r = 15 kpc = 4.629×10²⁰ m

v_baryon = √(6.674×10⁻¹¹ × 1×10⁴¹ / 4.629×10²⁰)
         = √(1.442×10¹⁰) = 1.20×10⁵ m/s = 120 km/s

v_MOND = 168 km/s (위에서 계산)

v_total = √(120² + 168²) = √(14,400 + 28,224) = 206 km/s
```

## 4️⃣ **암흑물질 수식**

### 기본 원리
```
v²_total = v²_baryon + v²_dark

v_baryon = √(G*M_baryon/r)
v_dark = √(G*M_dark(r)/r)
```

### NFW 프로파일 (Navarro-Frenk-White)
```
암흑물질 밀도 분포:
ρ(r) = ρ_s / [(r/r_s) * (1 + r/r_s)²]

ρ_s = 특성 밀도
r_s = 스케일 반지름

누적 질량:
M_dark(r) = 4π*ρ_s*r_s³ * [ln(1 + r/r_s) - (r/r_s)/(1 + r/r_s)]
```

### 계산 예시
```
파라미터 설정 (전형적 나선 은하):
ρ_s = 0.01 M_sun/pc³ = 4.08 × 10⁻²² kg/m³
r_s = 10 kpc = 3.086 × 10²⁰ m
M_baryon = 5 × 10¹⁰ M_sun = 1 × 10⁴¹ kg

위치: r = 15 kpc = 4.629 × 10²⁰ m

1) 바리온 속도:
v_baryon = √(G*M_baryon/r)
         = √(6.674×10⁻¹¹ × 1×10⁴¹ / 4.629×10²⁰)
         = √(1.442×10¹⁰) = 1.20×10⁵ m/s = 120 km/s

2) 암흑물질 질량:
x = r/r_s = 4.629×10²⁰ / 3.086×10²⁰ = 1.5

M_dark(r) = 4π × 4.08×10⁻²² × (3.086×10²⁰)³ × [ln(2.5) - 1.5/2.5]
          = 4π × 4.08×10⁻²² × 2.94×10⁶¹ × [0.916 - 0.6]
          = 1.51×10⁴¹ × 0.316
          = 4.77×10⁴⁰ kg

3) 암흑물질 속도:
v_dark = √(G*M_dark/r)
       = √(6.674×10⁻¹¹ × 4.77×10⁴⁰ / 4.629×10²⁰)
       = √(6.88×10⁹) = 8.29×10⁴ m/s = 83 km/s

4) 총 속도:
v_total = √(v_baryon² + v_dark²)
        = √(120² + 83²)
        = √(14,400 + 6,889)
        = √21,289 = 146 km/s
```

### 다른 거리에서 계산
```
r = 5 kpc:
  x = 0.5
  M_dark = 1.51×10⁴¹ × [ln(1.5) - 0.5/1.5] = 1.51×10⁴¹ × 0.072 = 1.09×10⁴⁰ kg
  v_baryon = √(6.674×10⁻¹¹ × 1×10⁴¹ / 1.543×10²⁰) = 208 km/s
  v_dark = √(6.674×10⁻¹¹ × 1.09×10⁴⁰ / 1.543×10²⁰) = 69 km/s
  v_total = √(208² + 69²) = 219 km/s

r = 30 kpc:
  x = 3.0
  M_dark = 1.51×10⁴¹ × [ln(4.0) - 3.0/4.0] = 1.51×10⁴¹ × 0.636 = 9.60×10⁴⁰ kg
  v_baryon = √(6.674×10⁻¹¹ × 1×10⁴¹ / 9.258×10²⁰) = 85 km/s
  v_dark = √(6.674×10⁻¹¹ × 9.60×10⁴⁰ / 9.258×10²⁰) = 83 km/s
  v_total = √(85² + 83²) = 119 km/s
```

## 5️⃣ **Loss 계산 수식**

### MSE (Mean Squared Error)
```
MSE = (1/n) * Σ(v_pred - v_obs)²

n = 데이터 개수
v_pred = 모델 예측값
v_obs = 관측값
```

### RMSE (Root Mean Squared Error)
```
RMSE = √MSE = √[(1/n) * Σ(v_pred - v_obs)²]

단위: km/s (속도와 같은 단위)
```

### MAE (Mean Absolute Error)
```
MAE = (1/n) * Σ|v_pred - v_obs|
```

### Chi-Squared (χ²)
```
χ² = Σ[(v_pred - v_obs)² / σ_obs²]

σ_obs = 관측 불확실성 (측정 오차)
```

### Reduced Chi-Squared
```
χ²_red = χ² / (n - p)

n = 데이터 개수
p = 자유 파라미터 개수

좋은 적합: χ²_red ≈ 1
```

### R² (결정 계수)
```
R² = 1 - SS_res / SS_tot

SS_res = Σ(v_obs - v_pred)²  (잔차 제곱합)
SS_tot = Σ(v_obs - v_avg)²   (총 제곱합)

범위: 0 ~ 1 (1에 가까울수록 좋음)
```

### Custom Loss (물리 제약 포함)
```
Loss = w₁*RMSE + w₂*Energy_Violation + w₃*Complexity

w₁, w₂, w₃ = 가중치
Complexity = 방정식 항의 개수 또는 깊이
```

## 6️⃣ **전체 계산 예시**

### 관측 데이터 (가상의 은하)
| 번호 | r (kpc) | v_obs (km/s) | σ (km/s) |
|------|---------|--------------|----------|
| 1 | 3 | 180 | 15 |
| 2 | 8 | 220 | 12 |
| 3 | 15 | 215 | 10 |
| 4 | 25 | 210 | 18 |
| 5 | 40 | 205 | 20 |

---

### **암흑물질 모델 (NFW)**

#### 파라미터
```
M_baryon = 5×10¹⁰ M_sun
ρ_s = 0.01 M_sun/pc³
r_s = 10 kpc
```

#### 예측값 계산
```
점 1 (r=3 kpc):
  v_baryon = √(G*M_baryon/r) = √(6.674×10⁻¹¹ × 1×10⁴¹ / 9.258×10¹⁹) = 268 km/s
  M_dark(3 kpc) ≈ 0.5×10⁴⁰ kg
  v_dark = 37 km/s
  v_pred = √(268² + 37²) = 271 km/s

점 2 (r=8 kpc):
  v_baryon = 164 km/s
  M_dark(8 kpc) ≈ 2.8×10⁴⁰ kg
  v_dark = 95 km/s
  v_pred = √(164² + 95²) = 189 km/s

점 3 (r=15 kpc):
  v_baryon = 120 km/s
  M_dark(15 kpc) ≈ 4.8×10⁴⁰ kg
  v_dark = 83 km/s
  v_pred = 146 km/s

점 4 (r=25 kpc):
  v_baryon = 93 km/s
  M_dark(25 kpc) ≈ 7.5×10⁴⁰ kg
  v_dark = 85 km/s
  v_pred = 122 km/s

점 5 (r=40 kpc):
  v_baryon = 73 km/s
  M_dark(40 kpc) ≈ 1.0×10⁴¹ kg
  v_dark = 78 km/s
  v_pred = 107 km/s
```

#### RMSE 계산
```
오차:
e₁ = 271 - 180 = +91
e₂ = 189 - 220 = -31
e₃ = 146 - 215 = -69
e₄ = 122 - 210 = -88
e₅ = 107 - 205 = -98

MSE = (91² + 31² + 69² + 88² + 98²) / 5
    = (8,281 + 961 + 4,761 + 7,744 + 9,604) / 5
    = 31,351 / 5
    = 6,270

RMSE = √6,270 = 79.2 km/s
```

#### χ² 계산
```
χ² = (91/15)² + (31/12)² + (69/10)² + (88/18)² + (98/20)²
   = 36.8 + 6.7 + 47.6 + 23.9 + 24.0
   = 139.0

χ²_red = 139.0 / (5 - 3) = 69.5  (매우 나쁨!)
```

#### R² 계산
```
v_avg = (180 + 220 + 215 + 210 + 205) / 5 = 206 km/s

SS_tot = (180-206)² + (220-206)² + (215-206)² + (210-206)² + (205-206)²
       = 676 + 196 + 81 + 16 + 1 = 970

SS_res = 91² + 31² + 69² + 88² + 98² = 31,351

R² = 1 - 31,351/970 = 1 - 32.3 = -31.3  (음수! 모델이 평균보다 나쁨)
```

---

### **MOND 모델**

#### 파라미터
```
M_baryon = 5×10¹⁰ M_sun
a₀ = 1.2×10⁻¹⁰ m/s²
```

#### 예측값 계산
```
v_MOND = (G*M*a₀)^(1/4) = 168 km/s (모든 외곽 영역)

점 1 (r=3 kpc, 내부):
  a_N = G*M/r² = 2.4×10⁻⁹ m/s² >> a₀
  뉴턴 영역: v = √(G*M/r) = 268 km/s

점 2 (r=8 kpc, 전이):
  a_N = 3.5×10⁻¹⁰ m/s² ≈ 3*a₀
  v ≈ 200 km/s (보간)

점 3-5 (r≥15 kpc, MOND 영역):
  v = 168 km/s
```

#### RMSE 계산
```
오차:
e₁ = 268 - 180 = +88
e₂ = 200 - 220 = -20
e₃ = 168 - 215 = -47
e₄ = 168 - 210 = -42
e₅ = 168 - 205 = -37

MSE = (88² + 20² + 47² + 42² + 37²) / 5
    = (7,744 + 400 + 2,209 + 1,764 + 1,369) / 5
    = 13,486 / 5
    = 2,697

RMSE = √2,697 = 51.9 km/s
```

#### χ² 계산
```
χ² = (88/15)² + (20/12)² + (47/10)² + (42/18)² + (37/20)²
   = 34.4 + 2.8 + 22.1 + 5.4 + 3.4
   = 68.1

χ²_red = 68.1 / (5 - 1) = 17.0  (여전히 나쁨)
```

#### R² 계산
```
SS_res = 13,486
R² = 1 - 13,486/970 = 1 - 13.9 = -12.9  (음수)
```

---

### **케플러 모델 (암흑물질 없음)**

#### 예측값
```
v = √(G*M_baryon/r)

점 1: v = 268 km/s
점 2: v = 164 km/s
점 3: v = 120 km/s
점 4: v = 93 km/s
점 5: v = 73 km/s
```

#### RMSE 계산
```
오차:
e₁ = 268 - 180 = +88
e₂ = 164 - 220 = -56
e₃ = 120 - 215 = -95
e₄ = 93 - 210 = -117
e₅ = 73 - 205 = -132

MSE = (88² + 56² + 95² + 117² + 132²) / 5
    = (7,744 + 3,136 + 9,025 + 13,689 + 17,424) / 5
    = 51,018 / 5
    = 10,204

RMSE = √10,204 = 101.0 km/s
```

## 💡 **핵심 정리**

### 좋은 모델의 기준
```
RMSE: 낮을수록 좋음 (< 20 km/s 우수)
χ²_red: 1 근처가 이상적 (0.5 ~ 2.0 허용)
R²: 1에 가까울수록 좋음 (> 0.9 우수)
Energy Violation: < 0.5 우수, < 2.0 허용
```

### 실제 연구에서
```
- 관측 오차 고려 (χ² 사용)
- 여러 은하 동시 분석
- 물리 법칙 제약 추가
- 베이지안 통계로 파라미터 추정
```
