## 2. ✨ Key Features

- **High-End UI**: Tailwind CSS 3.4 기반의 하이엔드 타이포그래피 및 Glassmorphism 레이아웃.
- **PNU Branding**: 부산대학교 공식 컬러 시스템(PNU Blue: #005BAA, Natural Gold: #B97C24) 완벽 적용.
- **Interactive Content**: 
  - **Hero Section**: 딥 블루 배경과 입자 애니메이션 효과.
  - **Vision & Mission**: 학과의 건학 정신과 교육 목표 시각화.
  - **Research Fields**: 입자, 응집물리, 핵물리, 응용물리 등 4대 핵심 연구 분야 인터랙티브 카드.
- **Responsive Design**: 모바일에서 울트라 와이드 모니터까지 완벽하게 대응하는 그리드 시스템.

## 3. 🛠 Tech Stack

- **Backend**: [FastAPI](https://fastapi.tiangolo.com/) - 고속 비동기 서버
- **Frontend**: [Tailwind CSS 3.4+](https://tailwindcss.com/) - 유틸리티 퍼스트 디자인
- **Templating**: [Jinja2](https://palletsprojects.com/p/jinja/) - 템플릿 엔진
- **Env Manager**: [uv](https://github.com/astral-sh/uv) - 고성능 패키지 관리

## 4. 🚀 Installation & Execution

### Quick Start
1. 서버 실행:
