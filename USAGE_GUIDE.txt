================================================================================
          GRAV (Galaxy Rotation Analyze Viewer) 정식 사용 설명서
================================================================================

본 프로그램은 은하 회전곡선 데이터를 바탕으로 AI가 직접 물리 수식을 추출하고,
뉴턴 역학, MOND, 암흑물질 할로 모델 중 어느 것이 가장 적합한지 판별하는 도구입니다.

1. 프로그램 시작 (Getting Started)
--------------------------------------------------------------------------------
(1) 배치 파일 실행
    - 바탕화면의 [GRAV_START.bat] 파일을 더블 클릭합니다.
    - 터미널에 로딩 화면이 나타난 후, 자동으로 데스크톱 UI 창이 실행됩니다.

(2) 데이터 선택
    - [Select Data Directory] 버튼을 눌러 은하 데이터(.dat 파일들)가 들어있는 폴더를 선택합니다.
    - 폴더 경로가 정상적으로 입력되면 아래에 폴더 이름이 표시됩니다.

2. 파라미터 설정 (Engine Configuration)
--------------------------------------------------------------------------------
(1) Neural Model
    - DeepONet: 복잡한 비선형 관계를 찾는 데 유리합니다.
    - FNO (Fourier Neural Operator): 물리 법칙의 전역적 특징을 추출하는 데 강점이 있습니다.

(2) Training Epochs (학습 횟수)
    - 권장값: 100 ~ 500
    - AI가 데이터를 얼마나 꼼꼼히 학습할지 결정합니다. 너무 낮으면 피팅이 엉성해집니다.

(3) PySR Iterations (수식 진화 횟수)
    - 권장값: 50 ~ 200
    - AI가 수식을 얼마나 정교하게 다듬을지 결정합니다. 100 이상이면 정밀한 수식을 찾아냅니다.

3. 분석 및 결과 확인
--------------------------------------------------------------------------------
(1) 분석 시작
    - [LAUNCH GRAV ENGINE] 버튼을 누르면 분석이 시작됩니다.
    - 왼쪽 하단의 'Real-time Output' 창에서 AI의 진행 상황을 실시간으로 확인할 수 있습니다.

(2) 대시보드 (격자 뷰)
    - 분석이 완료되면 우측 화면에 은하별 피팅 그래프가 격자 형태로 나타납니다.
    - 빨간 실선이 AI가 데이터로부터 직접 발견해낸 "새로운 물리 수식"의 결과입니다.

(3) 상세 분석 (팝업 창)
    - 격자에 있는 그래프 중 하나를 클릭하면 '상세 분석 팝업'이 뜹니다.
    - 여기서 해당 은하의 RMSE 수치와 발견된 최종 수식을 더 큰 그래프로 볼 수 있습니다.

4. 시스템 요구사항
--------------------------------------------------------------------------------
- OS: Windows 10/11 (64-bit)
- Python 3.9 이상 및 PyQt6 라이브러리 필요
- PySR 구동을 위한 Julia 환경 (최초 실행 시 자동 설치 시도)
